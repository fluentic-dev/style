`@fluentic/style` ships one package with isolated subpath exports for bundlers
and compiler integrations.

| Import | Purpose |
| --- | --- |
| `@fluentic/style/plugin/nextjs` | Next.js App Router integration |
| `@fluentic/style/plugin/vite` | Vite plugin |
| `@fluentic/style/plugin/webpack` | Webpack plugin |
| `@fluentic/style/plugin/rspack` | Rspack plugin |
| `@fluentic/style/plugin/farm` | Farm plugin |

Bundler adapters are intended for final app builds. Component libraries should
ship Fluentic source and let the consuming app plugin transform the package.

## Plugin options

```ts
stylePlugin({
  include: /\.[jt]sx?$/,
  exclude: /node_modules/,
});
```

See [Plugin Options](../../reference/plugin-options) for the full shape.

For App Router projects, see the [Next.js guide](../nextjs).
