Add the Vite plugin and configure React's automatic JSX runtime:

```ts title="vite.config.ts"
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';
import { plugin as stylePlugin } from '@fluentic/style/plugin/vite';

export default defineConfig({
  plugins: [
    stylePlugin({
      devSourcemap: 'sidecarServer',
    }),
    react({
      jsxImportSource: '@fluentic/style/plugin/jsx',
    }),
  ],
});
```

Set the JSX import source in TypeScript too:

```json title="tsconfig.json"
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "@fluentic/style/plugin/jsx"
  }
}
```

Vite is the smallest full setup path: write styles in TypeScript, extract
static CSS when possible, and keep runtime support for prop-driven or
theme-driven styles.

The plugin should run before `@vitejs/plugin-react` so Fluentic sees the source
before the React JSX transform. If you intentionally run without the plugin,
configure the style runtime yourself before rendering the app.

See [Plugin Options](options) for `include`, `exclude`, `devSourcemap`,
`getSourcemapFilePath`, and the development sourcemap modes.
