---
title: esbuild
description: Configure Fluentic Style with esbuild.
---

```js
import { plugin as stylePlugin } from '@fluentic/style/plugin/esbuild';
import { build } from 'esbuild';

await build({
  bundle: true,
  entryPoints: ['src/main.tsx'],
  format: 'esm',
  jsx: 'automatic',
  jsxImportSource: '@fluentic/style',
  loader: {
    '.ts': 'ts',
    '.tsx': 'tsx',
  },
  outfile: 'dist/bundle.js',
  plugins: [
    stylePlugin(),
  ],
});
```

Import both virtual modules once in your app entry:

```tsx
import 'virtual:fluentic-styles.css';
import 'virtual:fluentic-styles';
```

esbuild integrations are often explicit. Make sure your HTML and CSS assets are
copied or emitted by your build script.
