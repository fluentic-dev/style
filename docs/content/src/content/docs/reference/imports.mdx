---
title: Imports
description: Public subpath exports and when to use them.
---

| Import | Use |
| --- | --- |
| `@fluentic/style` | Runtime API, style builder, tokens, themes, and types. |
| `@fluentic/style/utils` | Utilities such as `exposeStyle`. |
| `@fluentic/style/selector` | Selector constructors, validators, presets, and priority helpers. |
| `@fluentic/style/css` | CSS helper APIs. |
| `@fluentic/style/dev/rsc` | Next.js App Router and RSC development bridge. |
| `@fluentic/style/plugin/jsx` | JSX runtime import source. |
| `@fluentic/style/plugin/vite` | Vite adapter. |
| `@fluentic/style/plugin/webpack` | Webpack adapter. |
| `@fluentic/style/plugin/rspack` | Rspack adapter. |
| `@fluentic/style/plugin/farm` | Farm adapter. |
| `@fluentic/style/plugin/parcel/config` | Parcel config. |
| `@fluentic/style/plugin/nextjs` | Next.js adapter. |
