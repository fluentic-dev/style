---
title: Tokens
description: createToken and createTokens create themeable value identities.
---

```ts
const accent = createToken('#2563eb');
```

```ts
const color = createTokens({
  accent: '#2563eb',
  danger: '#dc2626',
});
```

Use token calls to create overrides:

```ts
accent('#60a5fa');
color.danger('#ef4444');
```
