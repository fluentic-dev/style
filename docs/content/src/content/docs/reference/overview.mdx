Reference pages document exact public APIs.

Use Learn for the usage path, Beyond the Basics for advanced patterns, and
Design for architecture.

The main import is:

```ts
import { style, combineStyle, createTheme } from '@fluentic/style';
```

Other public subpaths are:

```txt
@fluentic/style/css       named CSS at-rule helpers
@fluentic/style/selector  custom selector builders and presets
@fluentic/style/utils     component-library utilities
@fluentic/style/dev       development utilities
@fluentic/style/dev/rsc   RSC development bridge
```

Each API page should answer:

```txt
what to import
what it accepts
what it returns
when to use it
where the guide lives
```
