---
title: JSX Runtime
description: Configure the css prop for React elements.
---

Fluentic Style ships a JSX runtime so DOM elements can receive a `css` prop:

| Prop | Accepts | Purpose |
| --- | --- | --- |
| `css` | resolved CSS items from `combineStyle`, arrays, falsy values | Adds generated class names. |

This prop is stripped before it reaches the DOM.

## Project-level setup

```json
{
  "compilerOptions": {
    "jsx": "react-jsx",
    "jsxImportSource": "@fluentic/style"
  }
}
```

Most React bundlers also need their React transform configured:

```ts
react({
  jsxImportSource: '@fluentic/style',
});
```

## File-level setup

```tsx
/** @jsxImportSource @fluentic/style */
```

This is useful when you want to adopt Fluentic Style gradually.

## `css` accepts arrays

```tsx
<button css={[css.root, isSelected && css.selected, props.css]} />
```

Falsy values are ignored. Nested arrays are supported.
