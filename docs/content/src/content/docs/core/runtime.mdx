The runtime is intentionally small. It turns style metadata into class names and
inserts rules only when they were not extracted ahead of time.

## `combineStyle`

```ts
const css = combineStyle(styles, ...targets);
```

`combineStyle`:

1. Normalizes scope target inputs.
2. Reuses a cached CSS instance for the same styles and scope targets.
3. Inserts runtime rules into the active sheet when CSS has not been extracted.

The JSX `css` prop can also accept direct `style(...)` and `style.slot(...)`
values. Use `combineStyle` when a component needs to carry scopes, slot
overrides, or token overrides along with its styles.

## Style sheets

The public sheet utilities include:

```ts
createStyleSheet
setGlobalSheet
```

Most apps do not need to touch these directly. They are useful when integrating
with SSR, tests, or custom runtimes.
